TestController - Следит за выполнением теста, делает выборку очередного вопроса из БД и т.д. Хранит свое состояние в TestState
TestState - Привязывается к определенному пользователю, хранит состояние теста(кол-во правильнык.неправильных ответов, номер текущего вопроса и т.д.)
Server - Сущность контролирующая исполнение теста и общение с пользователем.
